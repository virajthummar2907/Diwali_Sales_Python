#  Diwali Sales Analysis

##  Overview
This project analyzes Diwali sales data using Python to understand customer purchasing behavior and identify sales trends through data visualization.

##  Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

##  Analysis
- Data Cleaning
- Gender Analysis
- Age Group Analysis
- State-wise Sales
- Occupation Analysis
- Product Category Analysis

##  Key Insights
- Female customers spent more than male customers.
- Customers aged **26–35 years** made the highest purchases.
- Uttar Pradesh, Maharashtra, and Karnataka had the highest sales.
